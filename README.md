# math-learn-pdf

小学数学学习 PDF 资源仓库。收录人教版（五·四学制）数学教材扫描 PDF 及配套文本、知识点分析。

## 内容

| 文件 | 说明 |
|------|------|
| [`二年级上册数学.pdf`](./二年级上册数学.pdf) | 上海教育出版社（五·四学制）二年级上册，100 页扫描版 |
| [`二年级上册数学.txt`](./二年级上册数学.txt) | 经由 Apple Vision OCR 提取的完整文本 |
| [`大纲与知识点.md`](./大纲与知识点.md) | 全书 7 单元结构树 + 逐节知识点 + 能力图谱 |
| [`extract_text.swift`](./extract_text.swift) | 零依赖 OCR 提取脚本（macOS 专用） |
| [`拓展阅读.md`](./拓展阅读.md) | 与《十万个为什么（第六版）·数学》的逐单元关联映射 |

## 文本提取

macOS 用户可直接用系统「预览」选中文字（基于 Live Text）。如需批量导出文本：

```bash
swift extract_text.swift "二年级上册数学.pdf"
# 输出同目录下的 二年级上册数学.txt
```

脚本依赖 Apple Vision Framework（macOS Ventura+），无需安装第三方库。

## 添加新 PDF

按 `{年级}{上下册}数学.pdf` 格式命名，如 `三年级下册数学.pdf`。
