## 知识点总结

| 知识点 | 内容 |
|--------|------|
| 乘法的含义 | 几个相同加数相加 → 乘法表示 |
| 术语 | 乘数 × 乘数 = 积，× 叫乘号 |
| 读法 | 6×3=18 读作"6乘3等于18" |
| 交换律 | 交换乘数的位置，积不变 |
| 1 的乘法 | 任何数 × 1 = 本身 |
| 0 的乘法 | 任何数 × 0 = 0 |

## 亲子活动建议

### 活动1：找生活中的"相同加数"
- **筷子**：1双=2根，3双=2+2+2 → 3×2
- **车轮**：1辆小汽车=4个，3辆=4+4+4 → 3×4
- **鸡蛋**：每排5个×2排=10个 → 2×5

### 活动2：摆积木学交换律
用积木摆3×4的矩形，转90°就是4×3。数一数，结果一样。

### 活动3：读乘法算式
随手出题："读一读，3×4=12，哪个是乘数？哪个是积？"
