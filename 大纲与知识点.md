## 第1单元 · 100以内数的加减法（二）

### 1.1 不退位减法

- **核心方法**：个位减个位、十位减十位 → 合起来
  - 例：`35 − 2` → 先算 `5−2=3`，再算 `30+3=33`
  - 例：`58 − 25` → 先算个位 `8−5=3`，再算十位 `5−2=3` → `33`
- **竖式规则**：相同数位对齐，从个位算起
- **数位表**：用十位/个位表具象化操作（摆小圆片）
- **比一比**：`35−20` 与 `35−2` 的区别（减的是十位还是个位）

### 1.2 退位减法

- **核心概念**：个位不够减 → 从十位退 1 作 10
  - 例：`57−9` → 个位 `7−9` 不够 → 十位 5 退 1 剩 4 → `17−9=8` → `40+8=48`
  - 例：`43−18` → 个位 `3−8` 不够 → 十位 4 退 1 剩 3 → `13−8=5` → `30−10=20` → `20+5=25`
- **竖式标记**：退位点在十位数上标点
- **连续退位**：`40−26` → 十位 4 退 1 剩 3 → `10−6=4` → `30−20=10` → `14`
- **比较题型**：不退位 vs 退位的竖式区别

### 1.3 加与减（比多比少）

- **线段图**：用线段长短表示数量关系（引入工具）
  - `多多少`：大数 − 小数
  - `少多少`：大数 − 小数
  - `比一个数多几`：求较大数 → 较小数 + 差
  - `比一个数少几`：求较小数 → 较大数 − 差
- **标准表述**：
  - "鱼缸里有 18 条，桶里有 5 条，鱼缸比桶里多多少条？" → `18−5=13`
  - "红色彩纸比蓝色彩纸多 27 张，蓝色彩纸有多少张？" → 间接求较小数
  - "二年级比一年级多 18 人，一年级有多少人？" → 逆向思考

### 1.4 连减与加减混合

- **两步竖式**：`90−45−26` → 先算 `90−45=45`，再算 `45−26=19`
- **加减混合**：`39+38−37`、`38+24−18`、`43−7+16`
- **应用题模型**：
  - 总人数 − 女生 = 男生（`39+38−37`）
  - 总数 − 借出 = 剩余
  - 原有 − 借出 = 找回

## 第3单元 · 表内乘法

### 3.1 乘法引入

- **从加法到乘法**：几个相同加数相加 → 乘法表示
  - 例：`3+3+3+3+3+3=18` → `6×3=18`
  - 读作：6 乘 3 等于 18
- **术语**：乘数 × 乘数 = 积，`×` 叫乘号
- **交换律**：`3×4=4×3`（交换乘数位置积不变）
- **特殊数**：任何数 × 0 = 0，任何数 × 1 = 本身

### 3.4 乘加乘减

- **运算顺序**：先算乘法，再算加法或减法
  - 例：`5×4+3=23`（先算 `5×4=20`，再加 `3`）
  - 例：`6×4−1=23`（先算 `6×4=24`，再减 `1`）
- **简便思路**：把不足一组看成完整组再调整
- **常见题型**：
  - `3×2+4`、`12−2×5`、`7×8−13`
  - 判断够不够：`8×5=40`，班上有 39 人，40 > 39 → 坐得下

## 第5单元 · 分类

### 5.1 分类的标准

- **同一堆物品可按不同标准分类**
  - 按颜色：黄色 / 蓝色
  - 按形状：方的 / 圆的
  - 按扣眼数量：两眼 / 四眼
- **核心结论**：分类标准不同，分类结果也不同
- **表格整理**：统计每类的数量

### 5.2 逐层分类

- **多级分类**：先按颜色分 → 再按形状分 → 再按扣眼数量分
  - 黄色 13 个 → 方的 7 个 + 圆的 6 个
  - 蓝色 11 个 → 方的 7 个 + 圆的 4 个
  - 继续分至 12 小类
- **关键结论**：改变逐层分类的顺序，最终结果不变
- **实践活动**：用附页 3 的图形进行分类练习
